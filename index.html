This website

This is my first website! I've played around with the idea of building my own website for a while now. I've set up local Apache webservers before and built simple "Hello World"-pages before, but nothing serious.
The base for this website is the excellent template "Prologue" by HTML5UP. I took the the template as a starting point and just started working by adding content and modifying the stylesheet to my liking. The website is hosted on GitHub Pages, which is free and great for static webpages.
Check the GitHub repository

QuizPy/JS-Quiz

The project started as a simple command-line quiz written in Python. I wanted to host it online, but since I do not have a webserver at the moment and GitHub Pages only allows static content I rewrote the game in JavaScript. Python is used for parsing the questions and answers from uberspots OpenTriviaQA. I also set up a MySQL database (to store scores) and a small, free webserver at 000webhost.com.
